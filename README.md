# Applied ML Coursework Skill

A Claude Code skill for applied machine learning, data science, and data mining coursework.

This skill helps users:

- understand assignment briefs
- plan notebook workflows
- inspect datasets
- avoid data leakage
- implement machine learning and data mining algorithms
- evaluate model performance
- interpret results
- write academic reports
- check final deliverables

## Supported Coursework Types

- Classification
- Regression
- Clustering
- Association Rule Mining
- Text Mining
- Time Series Prediction
- Model Comparison
- Exploratory Data Analysis
- Data Preprocessing
